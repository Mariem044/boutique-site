
# StreetWear E-Commerce Platform

[![Lovable Project](https://img.shields.io/badge/Built%20with-Lovable-purple)](https://lovable.dev/projects/ff3ff883-e8b0-40e9-b55e-c39776ebfdea)

A modern, responsive e-commerce platform for streetwear fashion, built with React, TypeScript, and Tailwind CSS.

## 🚀 Demo

Try out the live demo: [StreetWear Demo](https://lovable.dev/projects/ff3ff883-e8b0-40e9-b55e-c39776ebfdea)

The demo showcases:
- Responsive design optimized for all devices
- Product browsing and filtering capabilities
- Product detail pages with interactive elements
- Shopping cart functionality
- User account management

## ✨ Features

- **Modern UI/UX**: Clean, intuitive interface with streetwear-inspired design elements
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop
- **Product Catalog**: Browse products by category, collection, or new arrivals
- **Shopping Cart**: Add, remove, and update quantities with real-time totals
- **User Accounts**: Sign up, sign in, and manage account details
- **Order Management**: Track orders and view order history
- **Wishlist**: Save favorite items for later

## 🛠️ Dashboard

The admin dashboard provides comprehensive tools for managing your e-commerce business:

- **Analytics**: Track sales, traffic, and customer behavior
- **Inventory Management**: Update stock levels and product information
- **Order Processing**: View, process, and fulfill customer orders
- **Customer Management**: Access customer information and purchase history
- **Content Management**: Update homepage banners, featured products, and promotional content

### Dashboard Access

To access the dashboard in the demo:
1. Navigate to `/admin` route
2. Use the demo credentials (username: `admin`, password: `demo123`)

## 🎨 Design System

StreetWear uses a custom design system with the following key components:

- **Color Palette**: Urban-inspired with vibrant accent colors
  - Primary: `#6b21a8` (Streetwear Purple)
  - Secondary: `#111827` (Dark Slate)
  - Accent: `#8b5cf6` (Vivid Purple)
  - Backgrounds: Various shades of gray and black
- **Typography**: Modern and bold with high readability
- **Components**: Reusable UI components built with shadcn/ui

## 💻 Tech Stack

- **Frontend**: React, TypeScript, Vite
- **Styling**: Tailwind CSS, shadcn/ui
- **State Management**: React Context API, TanStack Query
- **Animations**: Framer Motion
- **Routing**: React Router
- **Backend**: Supabase (authentication, database, storage)
- **Payments**: Stripe integration (coming soon)

## 🚀 Getting Started

### Prerequisites

- Node.js (v16+)
- npm or yarn

### Installation

```sh
# Clone the repository
git clone <repo-url>

# Navigate to project directory
cd streetwear-ecommerce

# Install dependencies
npm install

# Start development server
npm run dev
```

### Environment Setup

To connect with backend services, you'll need to set up environment variables through the Lovable interface or Supabase integration.
